Lost your token?

No worries! Generate a new one from the GoRest dashboard with these quick steps.

  1. 1
    Visit the official GoRest site and sign in with your account.
  2. 2
    Select Get Your Access Token in the dashboard to create a new personal token.
  3. 3
    Copy the generated token and paste it into the "Access token" field on this page.
