Hai perso il tuo token?

Nessun problema! Puoi generarne uno nuovo direttamente dal portale GoRest seguendo questi passaggi.

  1. Visita il sito ufficiale GoRest e accedi con il tuo account.
  2. Clicca su Get Your Access Token nella dashboard e genera un nuovo token personale.
  3. Copia il token creato e incollalo nel campo “Token di accesso” di questa pagina.
